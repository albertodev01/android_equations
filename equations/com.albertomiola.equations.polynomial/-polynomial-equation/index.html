PolynomialEquation

abstract class PolynomialEquation(val coefficients: List<Complex>)

Abstract class representing an polynomial equation, also know as algebraic equation, which has a single variable with a maximum degree.

The coefficients of the algebraic equations can be real numbers or complex numbers. These are examples of an algebraic equations of third degree:

  • x3 + 5x + 2 = 0

  • 2x3 + (6+i)x + 8i = 0

This class stores the coefficients list starting from the one with the highest degree.
