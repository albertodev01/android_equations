PolynomialDivisionResult

data class PolynomialDivisionResult(var quotient: PolynomialEquation, val remainder: PolynomialEquation)

This utility class holds the quotient and the remainder of a division between two polynomials. When you use PolynomialEquation.times, this class is returned. For example:

val numerator = Quadratic(1, -3, 2)
val denominator = Linear(1, 2)
val division = numerator / denominator // 'division' is of type 'AlgebraicDivision'
